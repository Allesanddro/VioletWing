Features

  • Automatic Trigger: Fires your weapon when an enemy is detected under your crosshair.
  • Configurable Trigger Key: Set a keyboard key or mouse button as the trigger.
  • Adjustable Delays: Customize shooting behavior with configurable delays.
  • Friendly Fire Toggle: Option to enable or disable firing at teammates.
  • Automatic Updates: Fetches the latest offsets and client data.
  • GUI Interface: Easy control and configuration through a graphical interface.

Installation

Option 1: Clone the Repository

  1. Ensure Python is installed on your system.
  2. Clone the repository: git clone https://github.com/Jesewe/cs2-triggerbot.git
  3. Navigate to the project directory: cd cs2-triggerbot
  4. Install the required dependencies: pip install -r requirements.txt

Option 2: Download Pre-Built Executable

Alternatively, you can download the ready-to-use executable from the Releases page. Simply download the latest version and run the executable directly.

Download Latest Release

Usage

  1. Run the main script: python main.py
  2. Configure the settings through the GUI as per your preferences.
  3. Launch Counter-Strike 2 and enjoy enhanced aiming assistance.
